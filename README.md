# 🩺 Insulin Webapp

A user-friendly, reliable web application designed to assist laypeople and the elderly in managing and tracking their insulin administration safely. This project aims to reduce errors in insulin dosing by providing an intuitive interface, offline capabilities, and timely reminders.

## 🚀 Features

* **Insulin Usage Logging:** A simple, error-resistant logging system for tracking daily insulin doses.
* **Permanent Offline Storage:** Built with **Hive** to ensure user data is safely retained locally on the device, even without an active internet connection.
* **Automated Alarm Services:** Background injection reminders to help users stick to their strict medical schedules.
* **Elderly-Friendly UI:** High-contrast elements, large touch targets, and clear typography tailored for accessibility.
